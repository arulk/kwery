Points to keep in mind while entering Parameter CSV:
  • First line of the CSV should contain parameter names
  • Subsequent lines should contain values
  • No need to quote values
  • If parameter values contain quotes, quotes should be escaped using double quotes, for example ""quoted value""
  • If you want to email report corresponding to a set of parameter values, you can do so using email_csv header
  • If email_csv has multiple emails, you need to quote it as "email1@company1.com, email2@company2.com"

If report template contains two sql queries select * from table0 where id = :id and select * from table1 where name = :name

Example Parameter CSV with email_csv:
id, name, email_csv
1, ram, "user0@company0.com, user1@company1.com"
2, ""guru"", "user2@company2.com, user3@company3.com"
3, vinod, user4@company4.com
Example Parameter CSV without email_csv:
id, name
1, ram
2, ""guru""
3, vinod
